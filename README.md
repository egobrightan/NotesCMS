# Notes CMS

## Functionality

- Adding/Managing notes
	- Printing notes
	- Support for wide screens (columns)
- Adding subjects (tabs)
	- From tabs bar or user profile
- Bookmarking
- Adding to favourites
- Basic user profile page
	- Themes
- Inverted colors mode
